Long Range Trajectory Generation

Project Course | 2020

In this course our group was provided an assignment by a company in the manufacturing industry. We were supposed to create a reference trajectory from point A to B that a differential drive robot could follow, given a certain factory layout. We created an algorithm capable of doing just that by combining a visibility graph of the environment with and a-star search algorithm. Furthermore, we created a feasible trajectory based on the discontinuous a-star path using non-linear model predictive control. The unpublished paper can be found here.

Reinforcement Learning Approach to Path Following

Project Course | 2020

As part of the Deep Machine Learning course at Chalmers University, me and my colleauge developed an reinforcement learning agent able to perform path following. While this is conventionally solved by letting a controller, e.g. a model predictive controller, calculate the throttle and steering angle, by giving our agent access to the location of future waypoints it was able to follow the predetermined path smoothly. The unpublished paper can be found here.

Development of a Quadrocopter Controlsystem

Steering and Motor Control in Autonomous Twizy

Bachelor Thesis | 2018

In our bachelor thesis, my and my group designed, implemented, and evaluated an electronic power steering system in a full-size Renault Twizy.
